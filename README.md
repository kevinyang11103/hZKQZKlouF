# 学生选课管理系统 java1165

## 项目概述

学生选课管理系统是基于Java + SpringBoot + Vue实现的前后端分离项目，旨在通过网络化、信息化的手段，提高学校选课工作的效率和准确性。

## 技术栈

- 架构：B/S、MVC
- 系统环境：Windows
- 开发环境：IDEA、JDK 17、MySQL
- 技术栈：Java、MySQL、SpringBoot、Mybatis、Vue

## 功能模块

### 系统角色

系统包含以下三个角色：

- 管理员（admin）
- 教师（teacher）
- 学生（student）

#### 管理员

- 实现对教师、学生、课程的增删改查（CRUD）操作
- 全方位控制教师业务和学生业务

#### 教师

- 查询自己开设的课程及选择这些课程的学生信息
- 登记学生成绩

#### 学生

- 实现选课和退课功能
- 实现成绩查询功能

## 运行环境

- 服务器：Windows 系统
- 数据库：MySQL
- 前端：Vue
- 后端：Java、SpringBoot
